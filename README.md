# TurnBaseGame

A grid-based turn-based strategy prototype developed with **Unreal Engine 5** and **C++**.

## Overview

TurnBaseGame is an early prototype of a tactical turn-based game built on a grid system.  
The project is focused on creating the core gameplay foundations, including map setup, selectable cells, unit placement, and future turn management.

## Tech Stack

- Unreal Engine 5
- C++
- Visual Studio 2022

## Current Features

- Grid-based game field
- Selectable cells
- Basic project structure in C++
- Initial camera setup
- Playable map prototype

## Planned Features

- Unit movement on the grid
- Turn system
- Enemy turns
- Combat system
- UI improvements
- Win / lose conditions

## Project Structure

- `Source/` — C++ gameplay code
- `Content/` — maps, materials, blueprints, assets
- `Config/` — project configuration files

## How to Open the Project

1. Clone the repository
2. Open `TurnBaseGame.uproject`
3. Generate Visual Studio project files if needed
4. Build and run the project through Unreal Engine or Visual Studio
